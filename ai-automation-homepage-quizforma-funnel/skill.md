# AI Automation Homepage + Quizforma Funnel Builder

## Purpose

Use this skill to create a high-converting homepage and lead funnel for an AI automation services business built around Open Claw installs, onboarding, hosting, support, and custom automation.

This skill is designed to position the offer around **business outcomes and done-for-you automation**, not cheap individual skills.

It must also create a quiz funnel, but the quiz itself must be created in **Quizforma**.

---

## Core Objective

Build a premium SaaS-style homepage that drives visitors toward one of these actions:

1. Claim a free installation
2. Request setup/onboarding
3. Take a qualification quiz in Quizforma
4. Book or request a custom quote

The page should emphasize:
- done-for-you setup
- business automation outcomes
- recurring hosting/support
- reduced manual work
- lead generation and workflow automation

---

## Mandatory Build Rules

### The bot MUST:
- create the homepage layout and copy
- create all section copy
- create CTA copy
- create a short homepage VSL script
- create a quiz funnel structure and question set
- prepare the quiz brief specifically for **Quizforma**
- create the results-page copy for each quiz outcome
- create the thank-you/next-step messaging
- create any short supporting microcopy needed for forms and buttons

### The bot MUST NOT:
- build the quiz as a native website component
- replace Quizforma with another quiz tool unless explicitly instructed
- position the offer primarily as a cheap skills marketplace
- lead with low-ticket skill pricing as the main homepage message

### The bot MUST treat Quizforma as:
- the platform used to create and host the quiz
- the source of the quiz experience
- the destination for the quiz flow when users click the quiz CTA

---

Position the business as a premium AI automation provider.

## VSL Deliverable

Write a 60–90 second homepage VSL.

### VSL structure:
1. Hook
2. Problem
3. Solution
4. What they get
5. Offer
6. CTA

### Tone:
- clear
- direct
- professional
- benefit-driven
- easy for non-technical business owners to understand

### Example content direction:
Hook:
What if your business could run key tasks automatically without you lifting a finger?

Problem:
Most business owners are stuck doing repetitive work every day.

Solution:
We install powerful AI automation systems directly into your business using Open Claw.

Offer:
We’ll install your system for free when you choose a hosting plan.

CTA:
Click below to get started.

The bot should write a polished final script, not just bullet points.

## Quizforma Quiz Build Instructions

The bot must create a complete quiz brief and content pack specifically for Quizforma.

### Important:
The bot is NOT building the quiz UI itself.
The bot is creating:
- quiz title
- subtitle
- question set
- answer options
- scoring or routing logic
- result pages
- CTA copy
- thank-you messaging
- embed instructions / handoff notes

The bot must present this as a **Quizforma-ready quiz blueprint**.

---

## Quiz Goal

The quiz must:
- qualify leads
- identify the most relevant automation type
- move visitors into a service-first mindset
- guide them toward setup/onboarding, not just DIY tools

---

## Quiz Title

Use:
What Should You Automate First in Your Business?

Optional alternate:
What’s Your Best AI Automation Setup?

---

## Quiz Subtitle

Create a short subtitle such as:
Answer a few quick questions to discover which automation system is the best fit for your business.

---

## Quiz Questions

Create these 5 questions with polished wording.

### Q1
What is your biggest bottleneck right now?
Options:
- Getting more leads
- Following up with leads consistently
- Managing customer inquiries
- Keeping internal workflows organized
- Saving time on repetitive admin tasks

### Q2
How are you currently handling this process?
Options:
- Completely manually
- With a mix of tools that don’t fully connect
- With some automation already in place
- We are not really handling it well right now

### Q3
How many leads, inquiries, or customer interactions do you manage each month?
Options:
- 0–10
- 10–50
- 50–200
- 200+

### Q4
What is your main goal with automation?
Options:
- Save time
- Increase leads
- Improve response speed
- Reduce missed follow-ups
- Scale operations more efficiently

### Q5
What level of help are you looking for?
Options:
- I want a fully done-for-you setup
- I want guidance and support
- I am comfortable with some DIY
- I’m not sure yet

---

## Quiz Outcome Categories

Create at least 4 result types:

1. Lead Capture & Follow-Up System
2. Customer Inquiry Automation System
3. Workflow & Operations Automation System
4. Done-For-You Growth System

The bot must write a unique result page for each.

---

## Result Page Format

For each result, create:
- result title
- short summary
- why this is the right fit
- key benefits
- recommended next step
- CTA button text
- optional microcopy

### Example CTA texts:
- Get This Installed For Me
- Book My Setup
- See My Best Plan
- Claim My Free Installation

---

## Suggested Quiz Logic

Use simple routing logic based on dominant answer themes.

### Example:
- leads + follow-up answers → Lead Capture & Follow-Up System
- customer inquiries answers → Customer Inquiry Automation System
- internal workflow/admin answers → Workflow & Operations Automation System
- strong done-for-you / scale intent → Done-For-You Growth System

The bot should format the logic clearly enough for implementation in Quizforma.
